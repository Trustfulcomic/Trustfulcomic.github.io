◆ GetClosestPoint()

wxPoint2DDouble GeoConic::GetClosestPoint ( const wxPoint2DDouble & pt) const
overridevirtual

Calculates the closest point on the curve to pt.

Parameters
ptThe point from which the distance is taken
Returns
The closest point
Note
This can have different behaviour from GetPerpPoint() on curves that are only part of a curve (such as segment). The function GetPerpPoint() "extends" the curve and gets the perpendicular projection. GetClosestPoint() returns the actual closest point.
